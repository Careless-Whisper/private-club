

<h1>Private club</h1>

<h2>Version de Ruby :</h2>
Ruby Version : 2.5.1

Rails Version : 5.2.0

<h2>Utilisation en local</h2>

git clone https://github.com/Careless-Whisper/private-club.git

cd private-club

bundle install --without production

rails db:migrate

rails db:seed

rails server

Go sur http://localhost:3000/


rails test

https://preevat-club.herokuapp.com/
